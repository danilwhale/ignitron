# installation

## obtaining

1. first of all, you need to get version of the game you currently have installed.
   to do this you can just open the game and look in the bottom left corner of the main menu
   or the title bar for version. both ways have it formatted in same way: `Allumeria Alpha x.y.z`,
   where x.y.z is the version you want.

2. go to [releases page of the modloader](https://github.com/danilwhale/ignitron/releases).
   release titles have supported game range in parenthesis, so look for top most release with target version matching yours.

3. just download the `Ignitron.Loader.zip` file located below the release description section.

## installing

### finding the game directory

> [!NOTE]
> this section is made for steam (playtest) releases, if you have an older version of the game (0.9.5 and below)
> you should already know where the game is located

1. open Steam
2. open 'Library' page in the top section of the window
3. press 'Allumeria Playtest' in the game list
4. look for a gear icon on the right side of the window and click it
5. open 'Manage'
6. open 'View local files'

### (actually) installing

1. create a 'mods' directory in the game directory you just found
2. remember `Ignitron.Loader.zip` you downloaded before?
3. extract it to the 'mods' directory you just created
